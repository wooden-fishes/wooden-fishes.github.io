INSTALL FAILED CONFLICTING PROVIDER

##出错信息
安装应用程序时出错：INSTALL FAILED CONFLICTING PROVIDER

##错误原因
多个程序使用了相同的provider，在AndroidManifest中配置了相同的android:authorities，在安装是其中一个程序时，再安装另外一个时，提示 provider冲突。在同一个设备上，provider配置的android:authorities必须是全局惟一的

##解决办法
在不同的应用中，配置不同的android:authorities

stackoverflow 有其它方案
http://stackoverflow.com/questions/16267785/install-shows-error-in-console-install-failed-conflicting-provider
