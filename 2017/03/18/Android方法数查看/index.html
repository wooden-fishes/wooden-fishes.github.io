Android 方法数查看

网站

APK method count
拖拽APK到网页上，稍后即可将其每个模块的方法数显示出来；

methodscount
可以搜索查出项目中使用的或要使用的Library的方法数，可以在引入Library进行查看，可以根据结果看是否要使
用此library；

工具

Classyshark
直接使用java jar 启用，可方便查看apk,dex等中每个包的方法数及使用了哪些开源库，并以饼状图展示出来，非
常直观；

dexdump是Android SDK自带的工具，位于build-tools目录下

插件

methodscount
这个是AS中的插件

dexcount-gradle-plugin
在Gradle中集成此library，在每次打包后都可以看到APK包的方法数比较方
